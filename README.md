# 🥗 Restaurante Saboroso

Site de restaurante desenvolvido durante o curso Técnico em Desenvolvimento de Sistemas.

## Sobre o projeto

O Restaurante Saboroso é um site criado para apresentar um restaurante e facilitar a interação dos clientes com o estabelecimento. Nele é possível conhecer o restaurante, visualizar o cardápio, consultar avaliações e realizar reservas.

O projeto foi desenvolvido com o objetivo de praticar conceitos de desenvolvimento web, organização de conteúdo, responsividade e experiência do usuário.

## Funcionalidades

- Apresentação do restaurante
- Exibição do cardápio
- Área para reservas
- Avaliações de clientes
- Informações de contato
- Layout responsivo

## Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript
